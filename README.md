This plugin allows simhub to simulate the 10 differents types of data when you're on the track.

## 🎯 Features

- Traction Control
  - TC level
  - TC Slip level
  - Power cut Level

- Engine Management
  - Engine map (Map +/-)
  - Energy recovery (Regen +/-)
  - Engine mixture (Mix +/-)

- Vehicle Dynamics
  - ABS management
  - Brake migration front/rear
  - Anti-roll bar front/rear

## 📥 Installation

1. Download TweaksOnTrack_1.0.0.zip
2. Extract TweaksOnTrack.dll
3. Copy the DLL file and the Languages folder to SimHub/ folder
4. Restart SimHub

## 🔧 Configuration

 - The commands will automatically appear in the SimHub interface. You can associate them in the command settings.
 - See the user manual page to configure them.
 - All data for your dashboard appears under [TweaksOnTrack] in SimHub.
## 🐛 Support

For any issue or suggestion, please create an issue on GitHub.


## 🐛 Know Issues
 - 🚧 **Coop Mode** : The values are not synchronised in coop mode. I am working on finding a solution to this issue. You can manually resynchronise by going to the maximum or minimum value in the game. Sorry for the    inconvenience.
  - 🔴 **MFD page**  : Changing settings from the MFD page is not supported.
